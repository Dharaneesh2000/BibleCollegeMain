## **5. WHY THESE PROVIDERS?**

### **Domain: BigRock/Hostinger**

**Why BigRock:**
- ₹9 for 3 years = cheapest option available
- ICANN accredited registrar
- Free WHOIS privacy protection
- Indian company with local support

**Why Hostinger:**
- Consistent ₹649/year pricing
- Global presence (150+ countries)
- Free privacy protection included
- Option to add hosting later if needed

**Why NOT GoDaddy:**
- Higher renewal costs (₹1,499/year)
- Confusing promotional pricing
- More expensive long-term

### **Hosting: Netlify**

**Why Netlify:**
- **FREE** for your current scale (100 users)
- React/SPA optimized (unlike traditional hosting)
- Global CDN included (fast worldwide)
- Automatic deployments from GitHub
- No server management required
- Scales automatically with traffic

**Why NOT Hostinger Business Plan:**
- ₹229/month = ₹2,748/year (expensive)
- Designed for WordPress, not React
- Requires manual server setup
- No automatic React build process

**Why NOT Vercel:**
- More expensive Pro plan (₹1,600 vs ₹1,520)
- Similar features to Netlify
- Slightly less generous free tier

### **Database: Supabase**

**Why Supabase:**
- **FREE** for 50,000 users (you need 100)
- PostgreSQL (industry standard)
- Built-in authentication system
- Row Level Security (enterprise-grade)
- File storage included
- Auto-generated APIs
- Scales automatically

**Why NOT Firebase:**
- Pay-per-read/write model can get expensive
- NoSQL (more complex for structured data)
- Unpredictable costs as you scale
- Less suitable for admin panels

**Why NOT AWS RDS:**
- No free tier
- Minimum ₹1,200/month
- Complex setup and management
- Requires DevOps expertise
